-------------------------------------------------
Readme for Ulg2Mat converter and logdata reshaper
-------------------------------------------------

Steps:

1. Mandatory step: Copy file *.ulg to same folder

2. Open Matlab2019b or newer

3. Enter "Ulg2Mat" into command line

4. Choose ulg file

5. Choose mat file title

-> A Logfile with raw data is created

To reshape Logdata into the same timeframe and easy accessable names. Following steps are necessary:

6. Enter "reshape_logdata" into command line

7. Choose the extracted logdata in .mat format

8. Choose name for reshaped logdata
